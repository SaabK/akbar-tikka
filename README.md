# Akbar Tikka Shop Website

A modern, responsive restaurant website built for **Akbar Tikka Shop, Gujranwala**.

The website showcases the restaurant's menu, gallery, special deals, business information, and provides an integrated online ordering experience through WhatsApp.

## Features

* Fully responsive design
* Mobile-first layout
* Fast-loading static website
* Online ordering system
* WhatsApp order integration
* Dynamic shopping cart
* Menu search and filtering
* Restaurant gallery with lightbox
* Testimonials section
* Contact and location information
* Google Maps integration placeholder
* Dark mode support
* FAQ section
* Floating WhatsApp button
* Call-now button
* Smooth scrolling animations
* SEO optimization
* Accessible design

## Tech Stack

* HTML5
* CSS3
* Vanilla JavaScript

No frameworks or external dependencies required.

## Customization

Before deployment, update:

* Restaurant phone number
* WhatsApp business number
* Address
* Google Maps embed link
* Menu items and pricing
* Social media links
* Gallery images

All placeholder sections are clearly commented in the code.

## WhatsApp Ordering

Orders are automatically formatted and sent to the restaurant through WhatsApp.

Each order includes:

* Unique Order ID
* Customer details
* Delivery address
* Ordered items
* Quantity
* Total amount
* Special instructions
* Date and time
